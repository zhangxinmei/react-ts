### install

```bash
  yarn
```

### dev

```bash
  yarn start:dev
```

> 静态资源打包后都在 **_build_** 目录下

### build:stage(stage 环境)

```bash
  yarn build:stage
```

### build:pre(pre 环境)

```bash
  yarn build:pre
```

### build:prod(prod 环境)

```bash
  yarn build:prod
```

### 支持

- vw 适配
- 多页
- 多环境(.env)
- typescript
- cra 2.x
- antd-mobile
- hooks
